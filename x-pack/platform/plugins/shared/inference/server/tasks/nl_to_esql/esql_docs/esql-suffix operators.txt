# SUFFIX OPERATORS

- `IS NULL`
- `IS NOT NULL`

### `IS NULL`

For NULL comparison, use the `IS NULL` and `IS NOT NULL` predicates.

**Example**
```esql
FROM employees
| WHERE birth_date IS NULL
```

### `IS NOT NULL`

For NULL comparison, use the `IS NULL` and `IS NOT NULL` predicates.

**Example**
```esql
FROM employees
| WHERE is_rehired IS NOT NULL
| STATS COUNT(emp_no)
```
