# UNARY OPERATORS

The only unary operator is negation (`-`):
**Syntax**
`UNARY OPERATORS()`

**Supported types**

| field         | result        |
|---------------|---------------|
| date_period   | date_period   |
| double        | double        |
| integer       | integer       |
| long          | long          |
| time_duration | time_duration |
