# TAN

Returns the tangent of an angle.

**Syntax**
`TAN(angle)`

### Parameters

#### `angle`

An angle, in radians. If `null`, the function returns `null`.

**Supported types**

| angle         | result |
|---------------|--------|
| double        | double |
| integer       | double |
| long          | double |
| unsigned_long | double |

**Example**
```esql
ROW a=1.8
| EVAL tan=TAN(a)
```

| a:double | tan:double         |
|----------|--------------------|
| 1.8      | -4.286261674628062 |
