# SPACE

Returns a string made of `number` spaces.

**Syntax**
`SPACE(number)`

### Parameters

#### `number`

Number of spaces in result.

**Supported types**

| number  | result  |
|---------|---------|
| integer | keyword |

**Example**
```esql
ROW message = CONCAT("Hello", SPACE(1), "World!");
```

| message:keyword |
|-----------------|
| Hello World!    |
