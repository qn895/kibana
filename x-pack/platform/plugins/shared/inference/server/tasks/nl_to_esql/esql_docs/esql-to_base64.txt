# TO_BASE64

Encode a string to a base64 string.

**Syntax**
`TO_BASE64(string)`

### Parameters

#### `string`

A string.

**Supported types**

| string  | result  |
|---------|---------|
| keyword | keyword |
| text    | keyword |

**Example**
```esql
ROW a = "elastic"
| EVAL e = TO_BASE64(a)
```

| a:keyword | e:keyword    |
|-----------|--------------|
| elastic   | ZWxhc3RpYw== |
