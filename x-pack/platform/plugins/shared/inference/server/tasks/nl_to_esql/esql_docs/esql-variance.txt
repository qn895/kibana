# VARIANCE

The population variance of a numeric field.

**Syntax**
`VARIANCE(number)`

### Parameters

#### `number`

**Supported types**

| number  | result |
|---------|--------|
| double  | double |
| integer | double |
| long    | double |

**Example**
```esql
FROM employees
| STATS var_height = VARIANCE(height)
```

| var_height:double |
|-------------------|
| 0.0425888         |
