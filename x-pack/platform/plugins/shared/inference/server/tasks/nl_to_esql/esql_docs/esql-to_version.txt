# TO_VERSION

Converts an input string to a version value.

**Syntax**
`TO_VERSION(field)`

### Parameters

#### `field`

Input value. The input can be a single- or multi-valued column or an expression.

**Supported types**

| field   | result  |
|---------|---------|
| keyword | version |
| text    | version |
| version | version |

**Example**
```esql
ROW v = TO_VERSION("1.2.3")
```

| v:version |
|-----------|
| 1.2.3     |
