# NOW

Returns current date and time.

**Syntax**
`NOW()`

**Parameters**
**Supported types**

| result |
|--------|
| date   |

**Examples**
```esql
ROW current_date = NOW()
```

| y:keyword |
|-----------|
| 20        |

To retrieve logs from the last hour:
```esql
FROM sample_data
| WHERE @timestamp > NOW() - 1 hour
```
