# SIN

Returns the sine of an angle.

**Syntax**
`SIN(angle)`

### Parameters

#### `angle`

An angle, in radians. If `null`, the function returns `null`.

**Supported types**

| angle         | result |
|---------------|--------|
| double        | double |
| integer       | double |
| long          | double |
| unsigned_long | double |

**Example**
```esql
ROW a=1.8
| EVAL sin=SIN(a)
```

| a:double | sin:double         |
|----------|--------------------|
| 1.8      | 0.9738476308781951 |
