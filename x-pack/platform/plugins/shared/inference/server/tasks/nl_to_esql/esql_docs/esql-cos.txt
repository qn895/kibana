# COS

Returns the cosine of an angle.

**Syntax**
`COS(angle)`

### Parameters

#### `angle`

An angle, in radians. If `null`, the function returns `null`.

**Supported types**

| angle         | result |
|---------------|--------|
| double        | double |
| integer       | double |
| long          | double |
| unsigned_long | double |

**Example**
```esql
ROW a=1.8
| EVAL cos=COS(a)
```

| a:double | cos:double          |
|----------|---------------------|
| 1.8      | -0.2272020946930871 |
