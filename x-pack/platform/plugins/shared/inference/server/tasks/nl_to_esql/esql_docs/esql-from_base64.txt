# FROM_BASE64

Decode a base64 string.

**Syntax**
`FROM_BASE64(string)`

### Parameters

#### `string`

A base64 string.

**Supported types**

| string  | result  |
|---------|---------|
| keyword | keyword |
| text    | keyword |

**Example**
```esql
ROW a = "ZWxhc3RpYw=="
| EVAL d = FROM_BASE64(a)
```

| a:keyword    | d:keyword |
|--------------|-----------|
| ZWxhc3RpYw== | elastic   |
